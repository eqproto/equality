## EQ Protocol — краткое описание

Трафик всегда разбит на фиксированные кадры одинакового размера и отправляется через одинаковые интервалы времени.

### Формат кадра
```
+---------+----------+---------------------------+
| type(1) | len(2)   | payload (len) + padding  |
+---------+----------+---------------------------+
```

Типы:
- 0 — AUTH
- 1 — DATA
- 2 — PADDING
- 3 — CLOSE

### AUTH
```
hash|timestamp|nonce|target
```

hash = SHA256(SID + ":" + timestamp + ":" + hex(nonce))

### DATA
Сырые данные.

### PADDING
Случайный мусор.

### CLOSE
Закрытие соединения.
